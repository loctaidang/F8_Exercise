Xin chào, mình là Minh Đăng!

Xuất phát điểm là một Backend Developer chuyên về Java, hiện mình đang trên hành trình mở rộng giới hạn bản thân để trở thành một Full-stack Developer. Không chỉ dừng lại ở việc xây dựng logic phía server, mình đang tích cực trau dồi các kỹ năng Frontend (HTML, CSS, JS) để có tư duy sản phẩm toàn diện và khả năng giải quyết vấn đề từ đầu đến cuối. Ngoài những giờ code căng thẳng, mình thường giải trí bằng game hoặc tìm tòi những công nghệ mới. Mình luôn sẵn sàng cho những thử thách mới và cơ hội hợp tác thú vị

Hành trình chuyển đổi

Mình tốt nghiệp với đồ án về ERP SAP. Mặc dù công nghệ này khác biệt, nhưng nó đã rèn luyện cho mình tư duy sắc bén về quy trình nghiệp vụ (Business Logic) trong các hệ thống doanh nghiệp lớn.

Nhận ra niềm đam mê nằm ở việc xây dựng các ứng dụng Web linh hoạt, mình đã kết hợp tư duy hệ thống chặt chẽ đó với kỹ năng lập trình hiện đại để tạo ra các sản phẩm không chỉ chạy tốt mà còn có cấu trúc vững vàng.

Kỹ năng chuyên môn

Dựa trên các dự án thực tế đã làm, bộ kỹ năng chính của mình bao gồm:

  • Ngôn ngữ & Framework: Java Core, Spring Boot (Spring Security, JPA/Hibernate).
  • Database: MySQL, SQL Server.
  • Công nghệ khác: Redis (Caching), WebSocket (Real-time), Firebase, Docker.
  • Kỹ năng mềm: Tư duy giải quyết vấn đề, làm việc nhóm (Git/Github).

Các dự án tiêu biểu

Dưới đây là các dự án mình đã tham gia xây dựng và đóng góp:

  1. Restaurant Management System (SWD392)
    Mô tả: Hệ thống quản lý nhà hàng hỗ trợ gọi món qua QR Code, quản lý đơn hàng và thanh toán.
    Công nghệ: Java Spring Boot, Redis (tối ưu hiệu năng), WebSocket (thông báo realtime), VNPay.
    Link GitHub

  2. MathchaEdu (SWP391)
    Mô tả: Nền tảng học toán trực tuyến (LMS) với các tính năng: Quản lý khóa học, bài kiểm tra (Quiz), theo dõi tiến độ học tập của sinh viên.
    Công nghệ: Spring Boot, Spring Security (Phân quyền User/Admin/Teacher).
    Link GitHub

  3. Chrono (EXE201)
    Mô tả: Ứng dụng quản lý công việc (Todo List) và năng suất cá nhân. Hỗ trợ tạo Workspace, quản lý Task theo khối thời gian.
    Công nghệ: Java Backend, thiết kế RESTful API.
    Link GitHub
